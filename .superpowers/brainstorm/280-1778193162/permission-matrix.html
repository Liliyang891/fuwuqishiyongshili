角色层级

等级 角色 AI 查询 文件操作 数据库操作
Lv.6 超级管理员 全部功能 + 系统配置 全部：读写删 + 上传 全部：CRUD + 建表删表
Lv.5 董事长 全部功能 + 查看审计 全部读写 / 不可删核心 SELECT 全部 / INSERT UPDATE

Lv.2 部门职员 受限 AI 查询（频率/范围） 本部门文件只读 / 可上传 仅 SELECT 本部门数据
Lv.1 游客 仅基础问答（不含敏感） 仅读公开文件 无数据库权限
