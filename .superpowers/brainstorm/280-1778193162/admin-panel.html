三个管理模块

用户管理

• 查看所有用户列表
• 筛选：按角色/部门/状态
• 编辑用户：修改角色和部门
• 启用/禁用账号
• 手动重置密码
• 删除用户（危险操作）

部门管理

• 查看部门列表
• 新增部门
• 修改部门名称
• 删除部门（需无用户归属）
• 查看各部门人数统计

审计日志

• 查看所有危险操作记录
• 筛选：按操作类型/用户/时间
• 记录：删除文件、DB DELETE
DROP TABLE、权限变更等
• 不可删除日志

新增 API 路由（全部仅超级管理员）

方法 路由 说明
GET/api/admin/users用户列表（支持筛选参数）
PUT/api/admin/users/{id}修改用户角色/部门/状态
POST/api/admin/users/{id}/reset-password重置用户密码
DELETE/api/admin/users/{id}删除用户
GET/api/admin/departments部门列表
POST/api/admin/departments新增部门
PUT/api/admin/departments/{id}修改部门名称
DELETE/api/admin/departments/{id}删除部门（需无用户）
GET/api/admin/audit-logs审计日志列表（支持筛选）
